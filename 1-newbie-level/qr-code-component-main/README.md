## Getting Started

### Prerequisites

- Node.js v24 or newer
- npm

### Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run start
```

The app will be available at [http://localhost:7979/](http://localhost:7979/).
